HTML5의 역사

HTML5 역사는 HTML 5 (이전의 철자가있는 HTML5 [a] )는 World Wide Web 에서 콘텐츠를 구성하고 표시하는 데 사용되는 마크 업 언어 입니다. HTML 표준 의 다섯 번째 현재 주요 버전이며 XHTML을 포함 합니다. 현재 두 개의 표준화 된 형식으로 존재합니다. HTML 5.2 권고 [4] 바이 월드 와이드 웹 컨소시엄 주로 웹 콘텐츠 개발자를 대상으로 (W3C, 조직의 폭 넓은 연합); 및 HTML 생활 수준 [5] 에 의해 WHATWG(네 개의 브라우저 공급 업체 중 소규모 컨소시엄)가 주로 브라우저 개발자 용으로 개발되었지만 요약 된 웹 개발자 버전에도 포함되어 있습니다. [6] 두 그룹의 명세 간에는 약간의 갈등이 있습니다.
